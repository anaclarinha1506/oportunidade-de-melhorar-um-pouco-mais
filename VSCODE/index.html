Bem-vindo ao meu espaço digital.

Aqui as ideias se tornam projetos. Este é um ambiente de testes criado para explorar design, estrutura e criatividade no desenvolvimento de front-end.

Sobre o projeto:

Esse projeto foi criado para meu processo de aprendizado em desenvolvimento de front-end. A proposta é experimentar layouts, testar componetes e entender como cada detalhe da interface influencia a experiência de quem navega.

A internet é contruída por milhões de pequenas decisões visuais e técnicas. Cada botão, cada cor e cada espaço em branco conta com uma história sobre como interagimos com a tecnologia.

O que estou aprendendo?

Estruturação de páginas HTM, estilização vizual com CSS, interatividade com JS e boas práticas de usabilidade e design.

Cada projeto é um portunidade de melhorar um pouco mais.
